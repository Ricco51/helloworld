### 二、空间变换

#### 1、基本概念

##### 图元表述（点线面）

| 点   | （x,y,z）                                               |
| ---- | ------------------------------------------------------- |
| 线   | x=(x2-x1) * u+x1 , y=(y2-y1)  * u+y1 , z=(z2-z1) * u+z1 |
| 面   | 点法式：向量(x-x1,y-y1,z-z1) * 向量N=0                  |

##### 空间计算

+ 飞机从坐标(x1,y1,z1)出发，沿(A,B,C)方向飞行一段距离D，到达坐标(x2,y2,z2)，求目标坐标

+ 求交：

|                | 直线         |
| -------------- | ------------ |
|                | **平面**     |
| **直线    与** | **二次曲线** |
| **平面    与** | **二次曲面** |
|                | **参数曲线** |
|                | **参数曲面** |

+ 方法：数学法，几何法
